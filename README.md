# pm-toolkit
Tools built to support a product-focused job search and demonstrate applied AI, automation, and workflow thinking.
# Resume Curator (/resume-curator)
A React artifact powered by the Claude API that tailors a master resume to a specific job description. Accepts a job description, company name, and role type (PM, APM, or PO), and returns a fully tailored resume with coaching notes on what changed and why. Built with the Anthropic Messages API and deployed as an interactive artifact in Claude.ai.
